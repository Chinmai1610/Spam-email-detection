# 📧 Spam Email Detection

A machine learning-powered web application designed to classify messages as **Spam** or **Ham (Not Spam)**. This project leverages Natural Language Processing (NLP) to analyze text patterns and provide real-time classification with a confidence score.


## 📊 Key Features

* **Real-time Prediction**: Instantly classify any text input into "Spam" or "Not Spam".
* **Confidence Scoring**: Provides a percentage indicating how certain the model is about its result (e.g., **92.62%**).
* **Minimalist UI**: Simple and clean web interface for ease of use.
* **NLP Integration**: Uses advanced text vectorization and machine learning algorithms for high accuracy.

## 📂 Repository Structure

The project follows a standard machine learning web app structure:

* **`data/`**: Contains the datasets used for training and testing.
* **`model/`**: Stores the serialized pre-trained machine learning model files.
* **`src/`**: Source code for the classification logic and backend.
* **`static/`**: CSS and assets for the web interface.
* **`templates/`**: HTML files for the frontend.
* **`requirements.txt`**: List of all Python dependencies required to run the project.

---

## 🛠️ Tech Stack

* **Frontend**: HTML5, CSS3
* **Backend**: Python (Flask/Streamlit)
* **Machine Learning**: Scikit-learn (Naive Bayes / SVM)
* **NLP**: NLTK / TF-IDF Vectorizer
* **Platform**: GitHub & Hugging Face Spaces
